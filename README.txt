Esse é um programa para verificar se as carteiras de motorista e as autorizações para dirigir do IBGE estão em dia,
Ao invés de procurar um a um, basta seguir seguintes os passos:

Digite os siapes dos funcionários no arquivo "siapes" separados por virgula, por exemplo, "1111111,2222222,3333333".
Coloque o arquivo excel do sistema na mesma pasta do arquivo principal.

Execute o programa
